# Currency converter

## Launch

Requires python 3.5 or higher

```bash
git clone git@github.com:n1te/converter.git .
pip install -r requirements.txt
python run.py
```

then go to http://localhost:8080

## Launch via docker

```bash
git clone git@github.com:n1te/converter.git .
./build.sh
```

then go to http://localhost

## Tests

```bash
python -m unittest -v
```
